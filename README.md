# guess_animal
Juego de consola en Python donde el jugador debe adivinar un animal a partir de pistas generadas dinámicamente por la API de OpenAI. Las pistas describen características del animal sin revelar su nombre, y el juego ofrece múltiples intentos para acertar.

Este proyecto es un juego de adivinanza de animales desarrollado en Python.

Cada partida selecciona un animal al azar de una lista.
El jugador ingresa su respuesta y recibe retroalimentación inmediata.
Si la respuesta es incorrecta, el juego proporciona pistas generadas automáticamente usando la API de OpenAI, describiendo características del animal sin mencionar su nombre.
La partida continúa hasta que el jugador acierte o se agoten los intentos.

Este proyecto sirve como ejemplo de integración de OpenAI en juegos interactivos y puede adaptarse para agregar más animales, niveles de dificultad, o interfaces gráficas.
